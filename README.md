# Card Voting Game (Node.js Version)

## 🚀 Overview
A card-based tournament game where players compete and win by votes.

## 🧱 Stack
- Node.js
- Express
- JSON file storage (no database)

## ▶️ Run

npm install  
npm start  

Open: http://localhost:3000

## 💾 Data
Stored in /data as JSON files.

## ⚠️ Notes
Not designed for production scale.
